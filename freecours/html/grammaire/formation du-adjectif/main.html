Formation du féminin

1. On a joute -e à la forme du masculin
  • grand / grande
  • petit / petite
  • blond / blonde
  • brun / brune
  • vert / verte
  • intelligent / intelligente
  • américain / américaine
  • marié / mariée
2. Les adjectifs qui finisseent en -e au masculin restent invirable
  • jeune / jeune
  • rouge / rouge
  • sympatique / sympatique
  • célibataire / célibataire
3. Les adjectifs qui finisseent en -en, -on ajoutent -ne
  • italien / italienne
  • ancien / ancienne
  • parisien / parisienne
  • européen / européenne
  • bon / bonne
  • mignon / mignonne
  • breton / bretonne
4. Les adjectifs qui finisseent en -el ajoutent -le
  • traditionel / traditionelle
  • exceptionel / exceptionelle
5. Les adjectifs en -er se transforment en -ère
  • fier / fière
  • étranger / étrangère
  • cher / chère
  • amer / amère
  • premier / première
6. Les adjectifs en -eux/eur changent en -euse
  • heureux / heureuse
  • rêveur / rêveuse
  • sérieur / sérieuse
  • nerveur / nerveuse
  • travailleur / travailleuse
7. Les adjectifs en -teur changent en -trise / teuse
  • conservateur / conservatrice
  • observateur / observatrice
  • menteur / menteuse
8. Les adjectifs en -f changent en -ve
  • sportif / sportive
  • actif / active
  • neuf / neuve
9. Les adjectifs en -c changent en -que
  • public / publique
  • turc / turque
  • grec / greque

Formation du pluriel

1. On ajoute un-s à la form singulier
  • grand / grands
  • intelligente / inteligentes
  • petit / petits
2. Les adjectifs qui finissent en -s / -x au singulier restent invariables
  • français / français
  • roux / roux
  • sérieux / sérieux
3. Les adjectifs qui finissent en -eau / -al au singulier prennent un -x au pluriel
  • nouveau / nouveaux
  • beau / beaux
  • original / originaux
  • sentimental / sentimentaux
