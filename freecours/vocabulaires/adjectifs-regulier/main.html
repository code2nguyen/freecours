Excercise du jour Exercice Vocabulaire Note personnelle Grammaire
La consonne finale est habituellement silencieux en adjectifs masculins
mais prononcées en adjectifs féminins
  • chaud / chaude
  • court / courte
  • droit / droite
  • fort / forte
  • froid / froid
  • gris / grise
  • haut / haute
  • laid / laide
  • lourd / lourde
  • mauvais / mauvaise
  • parfait / parfaite
  • petit / petite
  • sourd / sourde
Les terminaisons des adjectifs masculins sont suivants nasalisées.
Les terminaisons adjectif féminin ne sont pas nasalisées et la consonne finale prononcées.
  • amusant / amusante
  • content / contente
  • grand / grande
  • intelligent / intelligente
  • intéressant / intéressante
  • méchant / méchante
  • lent / lente
  • plein / pleine
  • prochain / prochaine
  • puissant / puissante
Les adjectifs suivants sont prononcés de la même
  • blue / bleue
  • clair / claire
  • compliqué / compliquée
  • désolé / désolée
  • dur / dure
  • égal / égale
  • fatigué / fatiguée
  • gai / gaie (happy)
  • joli / jolie
  • noir / noire
  • poli / polie
  • sûr / sûre
